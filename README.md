# 长离 (ChangLi)

智能视频管理播放平台，自动识别本地资源、智能分类、海报匹配、内置高清播放器。

**支持 macOS / Windows**

## 能力

- **智能识别** — 自动识别视频集、季、选集层级
- **分类管理** — 自定义分类、标签文件夹自动识别、一体化筛选
- **演员系统** — 演员信息管理、参演作品自动关联
- **海报匹配** — 自动匹配视频集和分集海报
- **内置播放器** — mpv 引擎，原片画质，倍速播放，画中画
- **更新维护** — 全量检查更新、内置下载、安装包清理
- **新手引导** — 首次启动步骤教程，覆盖主要功能

## 技术

Tauri 2.0 + React + TypeScript + mpv + SQLite
