# Changelog

## v1.7.19 (2026-07-01)

### 优化
- 合并 PR #24：将 Windows 自定义安装器外壳从 Win32 GDI 自绘升级为 Tao + Wry/WebView2 渲染，安装器 UI 改用 HTML/CSS 实现，提升圆角、阴影、渐变、字体层级和卡片质感上限
- 保留“自定义外壳 + 内层 NSIS 静默安装”架构：用户打开 `ChangLi_<version>_x64-setup.exe` 时看到 WebView2 品牌安装器，点击开始安装后通过 IPC 触发内层 NSIS `/S` 安装
- 安装器左侧保留 rose/orange 品牌渐变、真实 app 图标、品牌海报文案和能力 pill；右侧保留安装路径控制项、品牌 tint 能力卡、渐变进度条和高光主按钮

### 验证
- `npm run build`
- `cargo check --manifest-path installer-shell/Cargo.toml --target x86_64-pc-windows-msvc`

## v1.7.18 (2026-07-01)

### 修复
- 合并 PR #23：修复 Windows 自定义安装器窗口默认出现在屏幕左上角的问题，启动时按屏幕尺寸居中显示
