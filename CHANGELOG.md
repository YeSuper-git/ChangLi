# Changelog

## [1.9.16] - 2026-07-12

### Added
- **订阅编辑功能**: 每个订阅新增编辑按钮，可修改关联视频集和版本

### Fixed
- **启动/列表卡顿**: get_video_series_list不调用series_from_row避免读文件转base64，784视频集从卡1分钟变为秒开
- **选集删除卡顿**: 批量删除用单次IPC替代逐个调用
- **选集编辑按钮**: "批量删除"改为"编辑"
- **订阅检查更新匹配失败**: 统一过滤逻辑为prefix_part_matches_title，支持简中/繁中/简繁完整变体映射（简日内嵌、简体内嵌、简繁内封等）
- **参演作品海报不显示**: 空视频集和有视频的作品都从video_series单独读取poster_base64，不依赖SQL别名
- **滚动恢复失效**: 去掉navigationType依赖，兼容所有返回场景

## [1.9.15] - 2026-07-12

### Fixed
- **首页/演员页空白**: get_video_series_list SQL中s.has_actor和s.code列不存在于表中导致查询失败

## [1.9.14] - 2026-07-12

### Fixed
- **播放器堆损坏崩溃**: 避免前端和Rust端双重destroy，Rust端只hide，前端延迟800ms后destroy
- **列表操作卡顿**: get_video_series_list排除poster_base64字段，减少IPC传输量
- **订阅弹窗卡顿**: 新增轻量级查询get_video_series_list_lite，只加载id/title/display_type
- **订阅/标签页浮动按钮**: 改用FloatingActions组件，与首页/视频页/演员页一致
- **批量更新海报融合**: 融合重新生成海报缓存功能到批量更新按钮，移除独立按钮
- **订阅分组支持简日/繁日**: 支持嵌套方括号格式，简日优先级高于简繁
- **extract_season_number中文数字**: 支持一二三...等中文数字提取季号
- **extractVersionKey语言关键词**: 保留原始关键词(CHT/CHS)，修复前缀过滤匹配失败
- **mpv_send_command Windows兼容**: 添加空实现避免Windows编译失败
