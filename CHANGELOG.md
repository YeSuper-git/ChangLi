# Changelog

## [1.9.10] - 2026-07-12

### Fixed
- **macOS播放器语法错误**: Player.tsx else分支未闭合导致编译失败

## [1.9.9] - 2026-07-12

### Added
- **macOS播放器UI适配**: mpv通过IPC socket控制+属性轮询，前端控制栏正常渲染

## [1.9.8] - 2026-07-12

### Fixed
- **参演作品海报不展示**: 修复空视频集SQL缺少series_poster_base64别名
- **返回上一页滚动位置**: 简化逻辑用sessionStorage直接判断
- **订阅添加后立即显示**: 绑定后markDirty清除缓存
- **订阅检查更新季号匹配**: 支持RSS源标题提取季号，跨季不误判已有集
- **海报缓存重新生成**: 设置页添加重新生成海报缓存按钮
- **订阅/标签页刷新和返回顶部**: 标题栏添加刷新和返回顶部按钮
- **添加订阅弹窗缓存**: 每次打开重新挂载清空状态

## [1.9.7] - 2026-07-12

### Fixed
- **详情页海报不展示**: 改回用 poster_data_url (base64) 展示海报
- **海报分辨率提升**: poster_base64 从 300px 提高到 1200px，详情页更清晰
- **标签筛选不更新**: Tags 页保存后清除 Library 缓存
- **编辑态标签未按分类过滤**: 用 getTagsByCategory 按当前分类过滤标签
- **演员参演作品暂无资源不显示海报**: video_count=0 时显示占位图
- **返回上一页滚动位置恢复**: 用 useLocation 监听路径变化恢复滚动位置
