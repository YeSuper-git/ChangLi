# 长离 - 需求文档

## 项目信息

- **项目名称**: 长离 (ChangLi)
- **版本**: v1.8.x
- **日期**: 2026-07-10
- **类型**: 桌面应用

## 1. 项目背景

### 1.1 业务背景
用户需要一个智能视频管理播放平台，能够自动识别本地视频资源、智能分类管理、自动匹配海报、提供高质量播放体验。支持 macOS 和 Windows 跨平台使用。

### 1.2 技术背景
- 跨平台支持（macOS + Windows）
- 智能视频识别（季/选集/番号/海报）
- 高质量视频播放（原画质、原帧率）
- 自动化视频库维护（检查更新/自动改名/安装包清理）

## 2. 核心功能

### 2.1 智能视频识别
- 自动扫描本地目录，识别视频集、季、选集层级
- 支持 1-2季、1-3季 范围格式
- S01E01 标准季集格式
- 番号自动提取

### 2.2 分类与标签
- 自定义分类（动漫/泡面番/OVA 等）
- 标签文件夹自动识别
- 演员/标签/状态一体化筛选
- 标签支持双击编辑

### 2.3 演员管理
- 演员信息管理（生日/身高/体重/简介）
- 参演作品自动关联
- 隐藏彩蛋字段（三围/罩杯）

### 2.4 海报系统
- 自动匹配视频集海报（poster.jpg/cover.jpg/folder.jpg）
- 分集海报识别（优先当前文件夹，再往上一级）
- 编辑更换海报（本地上传/URL）

### 2.5 内置高清播放器
- mpv 播放引擎，原片原画质
- 倍速播放（0.25x-4x）
- 支持所有主流格式
- 画中画（PiP）模式

### 2.6 更新维护
- 全量检查更新（识别新增/移除/改名视频集）
- 单个视频集检查更新
- 内置下载更新包（显示进度，支持取消）
- 安装后自动清理安装包

### 2.7 新手引导
- 首次启动自动触发步骤引导式教程
- 覆盖主要功能页面
- 支持跳过/重新进入

## 3. 技术架构

### 3.1 技术栈
- 桌面框架：Tauri 2.0
- 前端：React + TypeScript + TailwindCSS
- 播放器：mpv (tauri-plugin-mpv)
- 数据库：SQLite
- 更新：GitHub Releases API

### 3.2 项目结构
```

├── src-tauri/          # Rust 后端
│   ├── src/
│   │   ├── main.rs     # 入口 + 命令

│   │   ├── player.rs   # 播放器引擎
│   │   └── db.rs       # 数据库操作
│   └── Cargo.toml
├── src/                # 前端
│   ├── components/     # 通用组件
│   ├── pages/          # 页面

```
