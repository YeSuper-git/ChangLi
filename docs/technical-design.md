# 长离 - 技术方案文档

## 1. 系统架构

### 1.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                      Tauri 2.0 桌面应用                       │
├─────────────────────────────────────────────────────────────┤
│  前端层 (React + TypeScript + TailwindCSS)                    │
│  ├── 首页：分类展示、最近观看                                 │
│  ├── 视频库：分类管理、标签筛选、全量检查更新                   │
│  ├── 视频集详情：选集列表、海报信息、关联演员                  │
│  ├── 演员库：演员管理、参演作品                               │
│  ├── 设置页：分类配置、标签管理、演员配置、检查更新             │
│  └── 播放器：mpv 内嵌播放窗口                                 │
├─────────────────────────────────────────────────────────────┤
│  Rust 后端层                                                 │
│  ├── scanner.rs：视频识别引擎                                │
│  ├── player.rs：播放器引擎                                   │
│  ├── db.rs：数据库操作 + 检查更新                            │
│  ├── main.rs：Tauri 命令入口                                 │
│  └── migrations.rs：数据库初始化 + 示例数据                   │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 数据库表

| 表名 | 用途 |
|------|------|
| video_series | 视频集（名称、路径、分类、海报） |
| video | 选集（路径、集数、观看进度） |
| actor | 演员信息 |
| actor_field | 演员字段定义 |
| actor_photo | 演员照片 |
| category | 分类配置 |
| tag | 标签 |
| series_tag | 视频集-标签关联 |
| series_actor | 视频集-演员关联 |
| play_history | 播放历史 |

### 1.3 关键技术点

#### 视频识别逻辑
1. 扫描分类根目录
2. 标签开启时：匹配标签名 → 进入标签目录找视频集
3. 不匹配标签 → 直接当视频集（4重去重）
4. 视频集内部：识别季/剧场版/直接选集

#### 检查更新
- 扫描本地文件夹 vs 数据库记录
- 新增：本地有但数据库没有 → 添加
- 移除：数据库有但本地没有 → 提示
- 改名：基名匹配但名称不同 → 自动更新

#### 播放器集成
- 使用 --wid 嵌入主窗口
- vo=gpu + hwdec=no（防崩溃）
- 微信输入法防注入

## 2. 前端页面

### 2.1 首页
- 分类列表（按最近观看排序）
- 追番标记、观看进度、连载状态
- 标签、演员、中文字幕（功能开关）

### 2.2 视频库
- 左侧分类导航
- 顶部筛选面板（标签/演员/状态）
- 视频集网格
- 右键菜单（移动/删除/追番/已看完）
- 全量检查更新按钮

### 2.3 视频集详情
- 海报 + 元信息
- 选集列表（支持编辑模式、拖动排序）
- 关联演员（可点击进入详情）
- 添加视频、检查更新

### 2.4 演员详情
- 头像 + 信息
- 参演作品列表
- 照片墙

### 2.5 设置页
- 网站管理
- 分类配置（扫描路径、功能开关）
- 标签管理（双击编辑）
- 演员配置（字段管理）
- 检查更新（内置下载）
- 新手引导按钮
